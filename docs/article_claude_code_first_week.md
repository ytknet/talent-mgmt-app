# Claude Codeを初心者から使いこなす！最初の1週間アクションまとめ

「AIでプログラミングができると聞いたけど、何から始めればいいかわからない」そんな方に向けて、Claude Codeを使い始めてから1週間で押さえておきたいアクションをまとめました。プログラミング未経験でも大丈夫です。一つひとつ順番に進めていきましょう。

---

## そもそもClaude Codeって何？

Claude CodeはAnthropic社が開発したAIアシスタント「Claude」をターミナル（コマンドライン）から使えるツールです。

難しく聞こえるかもしれませんが、要するに**チャットでAIに話しかけるだけで、コードを書いてくれたり、ファイルを編集してくれたりする魔法のツール**です。プログラミングの知識がなくても、「こんなものを作りたい」と日本語で伝えるだけで動くコードを生成してくれます。

---

## Day 1：インストールと初起動

まずはClaude Codeを使える状態にしましょう。

**やること**
- Node.js（無料）をパソコンにインストール
- ターミナルを開いて `npm install -g @anthropic-ai/claude-code` を実行
- `claude` とタイプして起動確認

**ポイント**
「ターミナルって何？」という方はMacなら「Launchpad」→「その他」→「ターミナル」で開けます。Windowsなら「PowerShell」を使いましょう。

起動したら「Hello！何ができますか？」と日本語で話しかけてみてください。それだけでOKです。

---

## Day 2：VS Codeと連携させる

Claude Codeは**VS Code（Visual Studio Code）というエディタと組み合わせると格段に便利**になります。VS Codeは世界中のエンジニアが使っている無料のコードエディタです。

**やること**
- VS Codeを公式サイトからダウンロード・インストール
- VS Code内のターミナルからClaude Codeを起動
- ファイルを開きながらAIと会話できることを確認

VS Codeと連携することで、AIが書いたコードをリアルタイムで画面上に表示しながら作業できます。「どこが変わったのか」が一目でわかるので、初心者にとって学習にもなります。

---

## Day 3：初めてのファイル作成をお願いする

環境が整ったら、実際にClaude Codeに作業をお願いしてみましょう。

**やること**
- 「index.htmlという自己紹介ページを作って」と頼んでみる
- 生成されたファイルをブラウザで開いて確認
- 「背景色を青にして」など、追加の指示を出してみる

**会話例**
```
あなた：「シンプルな自己紹介ページのHTMLファイルを作ってください」
Claude：「index.htmlを作成しました。名前や趣味を入力する欄があります」
あなた：「フォントをもう少し大きくしてください」
```

このように**普通の会話と同じ感覚で指示を出すだけ**でファイルが完成します。

---

## Day 4：よく使うコマンドを覚える

Claude Codeにはチャット以外にも便利なコマンドがあります。今日はよく使う3つだけ覚えましょう。

| コマンド | 何をするか |
|---------|-----------|
| `/help` | 使い方の一覧を表示する |
| `/clear` | 会話の履歴をリセットする |
| `/compact` | 長くなった会話を要約して軽くする |

**ポイント**
会話が長くなってくるとAIの応答が遅くなったり、話がずれてきたりします。そういうときは `/clear` や `/compact` を使うとスッキリします。新しいタスクを始めるときは `/clear` で一度リセットするのが習慣にするとよいでしょう。

---

## Day 5：設定をカスタマイズする

Claude Codeは自分好みに設定を変えることができます。

**やること**
- `/config` で設定画面を開く
- 応答言語を日本語に設定する
- プロジェクトフォルダに `CLAUDE.md` というファイルを作る

**CLAUDE.mdが重要な理由**

`CLAUDE.md` はClaude Codeへの「説明書」のようなファイルです。「このプロジェクトは〇〇のためのものです」「返答は必ず日本語で」などのルールを書いておくと、毎回同じ説明をしなくて済みます。

```
# CLAUDE.md の例

## プロジェクト概要
個人ブログのWebサイトです。

## ルール
- 返答は日本語でお願いします
- コードにはコメントをつけてください
```

---

## Day 6：エラーが出ても慌てない

プログラミングで一番つまずくのが「エラー」です。でもClaude Codeがあれば怖くありません。

**やること**
- エラーメッセージが出たらそのままコピーしてClaude Codeに貼り付ける
- 「このエラーはどういう意味ですか？」と聞く
- 「直してください」とお願いする

**会話例**
```
あなた：「このエラーが出ました→ SyntaxError: Unexpected token」
Claude：「このエラーは〇〇が原因です。△行目を修正しました」
```

エラーは「何か問題が起きたよ」というお知らせに過ぎません。Claude Codeに丸投げしてしまいましょう。

---

## Day 7：自分のプロジェクトを始める

1週間の締めくくりに、自分だけのプロジェクトをスタートさせましょう。

**おすすめの初プロジェクト例**
- 自己紹介ページを作る
- 毎日の学習記録をつけるメモアプリを作る
- 好きなものを紹介するWebページを作る

小さくていいです。「完成させた」という体験が次のモチベーションになります。

最初の1週間を乗り越えたあなたは、もうAIを活用できる人材です。
